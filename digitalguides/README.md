# WantWoTwee Digital Guides

Upload this entire folder beside the main website `index.html`. The page will be available at `/digitalguides/`.

## Before publishing

Open `guides.js` and replace:

- `6591234567` with your WhatsApp number, including country code but without `+`, spaces or dashes.
- `yourtelegramusername` with your Telegram username, without `@`.

## Add a guide

1. Add its WebP cover image to `assets/`.
2. Copy one guide object in `guides.js`.
3. Give it a unique `id` and update every field.
4. Add a new category name inside `categories` if needed. Filter buttons and counts are generated automatically.

Recommended cover size: 1200 × 1500 pixels (4:5), WebP format.

Payments are arranged manually through the enquiry conversation. Do not place bank details or sensitive payment information in these public files.
